# Moyo Budget App

## Project Overview
Moyo Budget is a simple yet powerful budgeting application that helps users manage their finances effectively. It offers functionalities for tracking expenses, setting budgets, and providing insights into spending habits.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Sigcino/BudgetTracker.git
   ```
2. Navigate to the project directory:
   ```bash
   cd BudgetTracker
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Features
- User authentication
- Expense tracking
- Budget setting
- Category management
- Monthly reports

## Architecture
The application follows a client-server architecture:
- **Frontend**: Developed using React for the user interface.
- **Backend**: Node.js with Express for the server-side logic.
- **Database**: MongoDB for data storage.

## Quick Start Guide
1. **Register/Login**: Create an account or log in to access your dashboard.
2. **Set a Budget**: Define your monthly budget to start tracking expenses.
3. **Track Expenses**: Add your daily expenses to monitor your spending.
4. **View Reports**: Use the reporting feature to analyze your financial habits and identify areas of improvement.
